# LDR-Controlled LEDs

A simple Arduino project where five LEDs are controlled according to the light intensity detected by an LDR sensor.

When the LDR reading reaches or exceeds the defined threshold, all LEDs turn on. Otherwise, they remain off.

The LDR is connected to analog pin `A5` through a voltage divider using a `10 kΩ` resistor.

The five LEDs are connected to digital pins `9`, `10`, `11`, `12`, and `13`. Each LED uses its own `220 Ω` current-limiting resistor.

## Components

- 1 Arduino Uno
- 1 breadboard
- 1 LDR sensor
- 5 LEDs
- 5 × 220 Ω resistors
- 1 × 10 kΩ resistor

## How it works

The Arduino continuously reads the voltage generated by the LDR voltage divider using `analogRead()`.

The analog reading ranges from `0` to `1023`.

In this project, the activation threshold is set to `800`.

- Reading greater than or equal to `800`: all LEDs turn on
- Reading below `800`: all LEDs turn off

The current LDR value is also displayed in the Serial Monitor every `100 ms`.

## Pin configuration

| Component | Arduino pin |
|---|---:|
| LDR voltage divider | A5 |
| LED 1 | 13 |
| LED 2 | 12 |
| LED 3 | 11 |
| LED 4 | 10 |
| LED 5 | 9 |
