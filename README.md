## About

This repository documents my learning journey with Arduino and basic electronics.

Each project is stored in its own folder and includes the source code, circuit details, and demonstrations whenever available.

The repository grows as I explore new components, programming concepts, and circuit-building techniques.

The purpose of this repository is to document my progress, practice building circuits, and strengthen my understanding of programming and electronics.
